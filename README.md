
# Electroshop

A online shop project with django



## API Reference

#### Run project

```bash
  ## Windows
  
  python3 -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
  python3 manage.py runserver


## Linux

  python3 -m venv venv
  source ./venv/Scripts/activate
  pip install -r requirements.txt
  python3 manage.py runserver
```
